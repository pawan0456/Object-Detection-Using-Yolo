# 🎯 Real-Time Object Detection using YOLOv8 + OpenCV

This project performs real-time object detection using a webcam.  
It uses the YOLOv8 Nano model from Ultralytics and OpenCV to detect objects and display bounding boxes with labels.

---

## 🚀 Features

- Real-time object detection from webcam
- Fast and lightweight YOLOv8 Nano model
- Detects multiple objects (COCO dataset)
- Displays bounding boxes with labels
- Press **Q** to exit

---

## 🛠️ Requirements

- Python 3.x
- OpenCV
- Ultralytics YOLOv8

Install dependencies:

```bash
pip install ultralytics opencv-python
